# Echo Abyss - Compact GDD

## Elevator Pitch

Echo Abyss is a top-down roguelike action RPG where each run is a dive into a comatose patient's mind. Players recover memory fragments for power, face manifestations of trauma, and make moral choices that shape patient outcomes and the story of NEXMIND.

## Core Pillars

1. Procedural psychological dungeons based on each patient
2. Build crafting through memory fragments and synergy
3. Moral choice tracking on multiple hidden axes
4. Narrative mystery with corporate conspiracy arc

## Loop

1. Hub at NEXMIND clinic
2. Choose patient and enter run
3. Clear rooms, gather fragments/relics, make choices
4. Resolve final abyss encounter
5. Return to hub with permanent meta resources

## Systems Included In Prototype

- 4-layer run structure
- HP + Sanity dual resource
- Active/passive fragment loadout
- Relic modifiers
- Morality axes:
  - Empathy vs Efficiency
  - Loyalty vs Rebellion
  - Preservation vs Extraction
- Pacify-or-fight final encounter logic

## Planned Expansion

- Procedural room graph and handcrafted room pools
- Additional enemy archetypes and bosses per patient
- Dialog trees and voiced memory scenes
- Interconnected lore archive across patients
- Audio reactivity by threat and sanity
